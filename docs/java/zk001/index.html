ZooKeeper

ZooKeeper#

简介：Apache ZooKeeper是一种用于分布式应用程序的高新更协调服务，提供一种集中式信息存储服务。

特点：数据存在内存中，类型文件系统的树形结构（文件和目录），高吞吐量和低延迟，集群高可靠

作用：基于zookeeper可以实现分布式统一配置中心、服务注册中心、分布式锁等功能。
