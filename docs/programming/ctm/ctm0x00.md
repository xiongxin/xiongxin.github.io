## 1.6 Correctness

如何验证程序正确性
- 需要对编程语言的操作进行数学建模，定义他们的操作。这个模型称为`language's semantics`
- 需要定义我们想要程序做什么。通常程序进行计算需要一个数学上定义的输入输出。这个称为 `program's specification`
- 我们使用数学技术推断程序，同时使用`semantics`。我们需要证明程序满足`specification`
